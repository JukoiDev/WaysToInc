
# WaysToInc

- Way 1a
```cpp
int i = 0;
i++; // Post-incrementation
```
- Way 1b
```cpp
int i = 0;
i = i++; // Post-incrementation
```
- Way 2a
```cpp
int i = 0;
++i; // Pre-incrementation
```
- Way 2b
```cpp
int i = 0;
i = ++i; // Pre-incrementation
```
- Way 3
```cpp
int i = 0;
i-= ~0; // Pro way
```
